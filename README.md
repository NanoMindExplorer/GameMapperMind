# Game Mapper

Aplikasi pemetaan gamepad (Keymapper) untuk menghubungkan kontroler fisik dengan layar sentuh Android, ditenagai oleh integrasi Shizuku.

## Fitur Utama
- **Visual Editor (WYSIWYG)**: Atur tata letak tombol, analog stick, dan area swipe secara langsung dan mudah.
- **Injeksi Sentuhan Tanpa Root**: Menggunakan Shizuku untuk mensimulasikan sentuhan layar yang responsif dan aman tanpa perlu root.
- **Aksi Swipe Cepat**: Dukungan untuk mengeksekusi gerakan swipe (Atas, Bawah, Kiri, Kanan) di layar hanya dengan menekan atau menahan satu tombol di gamepad fisik Anda.
- **Multi Profil**: Simpan dan kelola berbagai macam konfigurasi tombol untuk game yang berbeda-beda.

## Cara Menggunakan Fitur Swipe
Fitur **Swipe** dirancang untuk mengeksekusi gerakan menggeser di layar (misalnya untuk *skill moves* atau tipe tembakan khusus seperti *chip shot*) menggunakan satu tombol. 

Anda cukup memetakan aksi Swipe (misal: Swipe Kanan) ke salah satu tombol gamepad (misal: `X` atau `R-Stick`). Ketika tombol tersebut ditekan, sistem akan otomatis melakukan simulasi sentuhan menggeser ke arah yang ditentukan di dalam game.
